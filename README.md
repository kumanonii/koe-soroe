# こえそろえ（koe-soroe）

コラボ前に、**全員の声の大きさをそろえる**ツールです。

配信者がそれぞれ自分の配信に乗る音を確認し、コラボ相手の声が全員同じくらいの大きさで聞こえているかチェックします。「誰かだけ小さい」「一人だけ声が飛び出している」を、配信が始まる前に見つけて直せます。

ブラウザだけで動き、インストールも会員登録もいりません。データは外部に送信されません。

---

## できること

- コラボ相手の声が全員同じくらいの大きさか（バラつきがないか）をチェック
- 声が小さすぎ・大きすぎの人を判定し、配信者目線で調整方法をアドバイス
- 飛び出している人がいたら名前つきで警告（差4dB以上で検出）
- 静寂時のノイズ（雑音）もチェック
- 定型文＋お題の穴埋めで、誰でも自然に声を出せる

---

## このツールの役割とOBSの役割

| | やること |
|---|---|
| **このツール** | コラボ相手の声どうしを**揃える**（全員が同じくらいの大きさか確認） |
| **OBS** | 揃った全体を、**自分の声との相対バランスで調整**（自分が主役、相手は少し控えめに） |

---

## 使い方

### 1. 全員でページを開く

配信する人みんなが、同じURLをそれぞれのブラウザ（**Chrome / Edge 推奨**）で開きます。配信しないメンバーには画面共有で指示を見せながらチェックしましょう。

### 2. メンバーの名前を入れる

コラボに参加する人の名前を入力します。

### 3. Discordの音声を取り込む

「Discordの音声を取り込む」を押し、共有画面で**「システム音声（音声も共有）」をオンにして**共有します。「Chromeタブ」「ウィンドウ」「画面全体」のどれを選んでもOKです。

> ⚠ どの方法でも**PC全体の音**（通知音・BGM・ゲーム音など）が取り込まれます。チェック中はDiscord以外の音を止めるかミュートにしてください。

> ⚠ 共有の選択画面にはタブ名やウィンドウ名が表示されます。Discordで画面共有中の場合は選択画面も相手に見えるので、**音声の取り込みはDiscordの画面共有を始める前に**済ませておきましょう。

### 4. お題を選ぶ

チェック画面でお題のカテゴリ（食べ物・動物・ゲームなど）を選びます。**全員で同じものを選んでください。**

選んだカテゴリに合わせて「こんにちは、〇〇です。好きな○○は〇〇です！今日はよろしくお願いします！」のテンプレートが表示されます。〇〇に自分の答えを入れて読み上げるだけです。

### 5. ひとりずつチェック

チェックする人を決め、**全員で「せーの」でその人の「開始」ボタンを一斉に押します。**

1. **3・2・1** のカウントダウン
2. **3秒だけ黙る** → ノイズをチェック
3. 合図が出たら **テンプレートを読み上げる** → 声の大きさをチェック

### 6. 全員ぶん終わったら完了

結果画面で、全員の声が揃っているか・飛び出している人がいないかを確認します。調整が必要な人がいたら、Discordの個別音量で調整してやり直しましょう。

声が揃ったら、あとはOBSで「自分の声が一番大きく、コラボ相手はそれより少し小さく」なるよう全体を調整すれば完了です。

## 動作環境

- **ブラウザ：Chrome / Edge を推奨**（Firefox やスマホ版は音声共有に対応していないことがあります）
- パソコンでの利用を想定しています

---

## うまくいかないとき

| 症状 | 対処 |
|------|------|
| メーターが動かない | 共有時に「システム音声を共有」のチェックを入れ忘れていないか確認。もう一度「取り込む」を押してやり直せます。 |
| 音声共有の選択肢が出ない | Chrome / Edge を使ってください。 |
| 自分の声が0のまま | 正常です。自分の声は自分のDiscordから返らないため測れません。他の人の画面で確認してもらいましょう。 |

## セキュリティ

- このツールは画面共有の**音声だけ**を使用し、映像は読み取っていません
- すべてのデータはブラウザ内でのみ処理され、外部のサーバーへの送信は行っていません
- ソースコードはこのリポジトリで公開されています
