# こえそろえ（koe-soroe）

コラボ配信を始める前に、**参加者全員の声の大きさをそろえる**ための音量チェックツールです。

「マイクが小さいまま配信が終わってしまった」「誰かだけ音が割れていた」を、始める前になくします。ブラウザだけで動き、インストールも会員登録もいりません。

---

## できること

- 参加メンバーを並べて、ひとりずつ声の大きさをチェック
- 「いつもの声」と「少し大きめの声」の2段階で測定
- 各自の音量を **−12dB 〜 −6dB の適正範囲** に収まっているか判定
- 静かにしている間の **ノイズ（雑音）** もチェック
- 結果に応じて「✓ OK / ⚠ 要調整 / ✕ 音声未検出」を色分け表示

---

## 使い方

### 1. 全員でページを開く

コラボに参加する人**全員**が、同じURLをそれぞれのブラウザ（**Chrome または Edge 推奨**）で開きます。

> このツールは「**ボタンを押した人が、その瞬間に聞いている声**」を測ります。だから2人以上で開いて、お互いの声を測り合うのが基本です。自分の声は自分では測れません。

### 2. メンバーの名前を入れる

参加者の名前を入力します（自分を含めてOK）。

### 3. Discordの音声を取り込む

「Discordの音声を取り込む」を押し、共有画面で **「画面全体」を選び、「システム音声（音声も共有）」にチェック** を入れて共有します。

> ⚠ **「ウィンドウ」でアプリ単体を選ぶと音声が取り込めないことがあります。** 確実なのは「画面全体」です。

### 4. ひとりずつチェック

チェックする人を決め、**全員で「せーの」でその人の「開始」ボタンを一斉に押します。**

> 各自バラバラに押すと測るタイミングがズレてしまうため、**一斉に押す**のがコツです。

押すと次の流れで進みます：

1. **3・2・1** のカウントダウン
2. **3秒だけ黙る** → ノイズをチェック
3. 合図が出たら、**その人だけが画面の文章を読み上げる** → 声の大きさをチェック

これを「いつもの声」「少し大きめの声」の2回おこないます。

### 5. 全員ぶん終わったら完了

全員が適正範囲に収まれば、音量そろえ完了です。小さい人・大きい人がいたら、その場でマイク音量を調整してもう一度チェックしましょう。

---

## ひとりで自分の声を確認したいとき

Discordの設定 →「音声・ビデオ」→ マイクテストの「確認してみよう」をオンにすると、自分の声が返ってくる状態になり、自分の声も測れます。

> ※テスト中はDiscordの通話が一時的に止まります。コラボ前のひとり確認用です。最終確認は他の人にも見てもらうのが確実です。

---

## 動作環境

- **ブラウザ：Chrome / Edge を推奨**（Firefox やスマホ版は音声共有に対応していないことがあります）
- パソコンでの利用を想定しています

---

## うまくいかないとき

| 症状 | 対処 |
|------|------|
| メーターが動かない | 共有時に「システム音声を共有」のチェックを入れ忘れていないか確認。「画面全体」を選ぶのが確実です。 |
| 音声共有の選択肢が出ない | Chrome / Edge を使ってください。 |
| 自分の声が測れない | 正常です。自分の声は自分のDiscordから返らないため、他の人に見てもらってください。 |
